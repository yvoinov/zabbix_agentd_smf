***   ВНИМАНИЕ!    Пакет    по   умолчанию   настроен  на  запуск
zabbix_agentd, установленный в /usr/local!

Данный   набор  скриптов  предназначен  для  установки и удаления
сервиса SMF для zabbix_agentd на Sun Solaris 10 и выше.

Все  необходимые  пререквизиты,  необходимые  для запуска сервиса
zabbix_agentd, согласно руководству по конфигурированию (исключая
конфигурирование собственно zabbix_agentd) выполняются в процессе
установки   сервиса,   удаление    сервиса    SMF    удаляет  все
скрипты  и  выполняет  разрегистрацию сервиса zabbix_agentd и его
остановку.

Для работы сервиса по умолчанию должен быть создан пользователь и
группа:

groupadd zabbix
useradd -g zabbix -d /usr/local/sbin -s /bin/sh -c "Zabbix Monitoring System" zabbix
passwd -N zabbix

Для активизации zabbix_agentd SMF выполните следующие шаги:
-----------------------------------------------------------

1. Установите zabbix_agentd и все необходимые библиотеки,если это
еще не сделано.

2.       Сконфигурируйте       zabbix_agentd      редактированием
zabbix_agentd.conf.

3.     Запустите    скрипт   zabbix_agentd_smf_inst.sh,   который
и  выполнит       все       необходимые      пререквизиты       и
постинсталляционные     действия    и   установит   zabbix_agentd
SMF сервис.

4.  Выполните команду svcadm enable zabbix_agentd для активизации
и старта сервиса zabbix_agentd.

Для деактивации и удаления zabbix_agentd SMF сервиса выполните
--------------------------------------------------------------
следующие шаги:
---------------

1.Запустите скрипт zabbix_agentd_smf_rmv.sh. Скрипт останавливает
все   работающие   процессы  zabbix_agentd,  рарзегистрирует  SMF
сервис,   полностью   удаляет   zabbix_agentd  SMF  и  откатывает
операции, выполненные ранее при активизации сервиса.

***    Замечание:     Удаление   сервиса  zabbix_agentd   SMF  не
приводит к удалению установленного ПО zabbix_agentd, пользователя
и группы.

Для удаления пользователя и группы выполните:

userdel zabbix
groupdel zabbix
